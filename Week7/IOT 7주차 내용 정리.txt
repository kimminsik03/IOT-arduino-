패킷 - 프로토콜에 맞춰서 여러개의 바이트를 전송  (네트워크에서 사용 )
데이터 통신 - 2비트 데이터를 한 비트씩 전송 (아두이노에서 사용)


전송속도 

비트레이트 
	초당 전송 가능한 비트 수 단위는 bps
	오늘날 대역폭은 75Mbps 유선은 Gbps급 속도  

보 레이트 
	초당 전송 가능한 펄스 또는 심볼의 수 단위 baud

대역폭 
	데이터를 전송하는 데 필요한 주파수폭을 말함
	넓으면 그만큼 더 많은 데이터 전송 가능 


유선통신 케이블 
	
	동축 케이블 
		내부도체부터 외부의 플라스틱 외피까지 모두 중심축이 같다는 의미 
	
	이중와선 
		케이블이 길어져 외부 통신의 영향의 영향을 막기 위해 두개 선이 꼬아져 있음 
		LAN 근거리 통신망에서 사용 
	
	광 케이블 
		빛으로 데이터 전송 
		싱글모드- 하나의 빛이 목적지까지 신호 전송 중장거리용
		멀티모드 - LED로 빛확산 싱글모드보다 전송거리 짧아짐 

무선통신 
	3000Hz~3000GHz까지의 전자기파를 의미 
	주파수는 높은 곳에서부터 낮은곳까지의 1초에 몇 번 반복 하는지 단위 Hz
	파장은 높은 곳 -> 높은 곳, 낮은 곳 -> 낮은 곳 사이의 거리 
	
	주파수가 높으면 많은 정보 전송, 장애물 간섭많아짐 
	낮으면 직진성은 약하지만 넓은 지역 송신 유리, 정보량 적음  
 

직렬 통신 / 병렬 통신

	직렬 통신 
		데이터가 일렬로 전송 /1차선 도로 (고속통신 유리)

	병렬 통신
		여러데이터가 동신에 전송 /8차선 도로 


병렬 통신 단점 
	클록이 틀어지는 현상이 발생 
	-> 비트 정렬 힘들어짐 


단방향 통신과 양방향 통신 
	단방향 통신 
		-신호가 한쪽으로 전송 - 리모콘
	
	양방향 통신
		-신호가 양쪽으로 전송 - 휴대폰 전화
	
	전이중 통신 
		- 언제든지 송신과 수신이 동시에 가능 -이동통신 LTE
	
	반이중 통신 
		-송신과 수신이 한 번에 하나씩만 가능 -무전기 

동기식/ 비동기식 통신 

	동기식식 통신 (신뢰성 ㅇ 속도 느림 )  -TCP
		- 통신을 위해 클럭을 맞춤
	
	비동기식 통신 (신뢰성 x, 속도 빠름)  - UDP
		송,수신측 합의 없이 데이터 전송 

다중화 
	여러 신호를 합쳐 하나의 신호로 만든 후 전송 

	주파수 분할 다중화(FDM)
		-주파수를 나누어 전송 
	시분할 다중화(TDM)
		-시간을 나누어 전송 

데이터 통신 방식에 따른 네트워크 분류
	
	유니캐스트 
		-송수신이 하나씩 존재 
	멀티캐스트 
		-송신 1개 수신 여러개 
	브로드캐스트 	
		-송신 하나  수신은 네트워크에 있는 모든 노드

		홈 네트워크(HAN)

회선 교환망/ 패킷 교환망

		전화망에서 사용 여러 대의 교환기를 거쳐서 상대방 전화기와 연결
	
	패킷 교환망

토폴리지 
	네트워크를 요소를 물리적으로 연결한 것 
	
	종류 
		스타 토폴리지 - 별처럼 생김 중앙이 끊기면 다 끊김
		트리 토폴리지 - 최하단 컴퓨터 상단에 교환기장비 (계층구조 )
		버스 토폴리지 - 하나의 선에 여러 요소가 붙어 있음  - 요소 추가 쉬움 
		링 토폴리지 - 원형으로 구성 - 요소 추가 어려움 , 버스에 밀림 
		매쉬 토폴리지 - 그물망처럼 이어져 있음 - 하나가 끊어져도 나머지로 연결 가능 
		(인터넷에서 사용되는 토폴리지는 매쉬 토폴리지)
