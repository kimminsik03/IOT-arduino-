빅데이터 이론 강의 

IOT랑 빅데이터 기술이 비슷
사물인터넷 - 데이터 센싱 초점
빅데이터 - 대량의 데이터에 초점

빅 데이터 의미
	대량의 	
	정형 -> 규칙이 정해져 있는 데이터   EX)ACCESS,EXCEL 
	반정형 -> 규칙이 반만 정해져 있는 데이터 EX) 시계열 DB, HTML
	비정형 -> 정보만 있는 날 것의 데이터 
	데이터를 가지고 유요한 정보를 도출하는 것

속성 (3V)
	규모 
		-> 제타 바이트 시대에 진입 
	다양성 
		-> IOT로 인해(센싱 기술) 데이터의 종류가 많아짐
	속도 
		-> 실시간 처리에서 언급
		->신뢰성, 가치정확성 휘발성(실시간 데이터가 무의미한 데이터 생김)이 추가되어서 6V가됨  
	
빅데이터 분석의 필요성 
	-인터넷 사용으로 인한 데이터 폭증 
	-비정형 데이터를 정형화 분석하려는 시도중 
	

빅데이터 처리 과정 (읽어보기)
	-수집
		-내부데이터 -> 기존에 있는 정형의 데이터
		-외부데이터 -> 인터넷에 있는 비정형 데이터 
		=> 외부,내부데이터 차이 = 정형,비정형 차이 
	-저장
		=>비정형 데이터를 정형 데이터로 저장하는 기술이 필요 
	-처리 
		=>필요없는 데이터는 제거 남은 데이터를 효율적으로 재배치함 
	-분석
		=>다양한 알고리즘을 통해 분석 결과를 만들어냄 
	-표현	
		=> 변환된 정형데이터를 사람이 인지할 수 있는 형태로 표현 

빅데이터 기술
	로그 수집기 => 내부 웹서버에서 로그 수집
	크롤링 => 봇을 사용해 인터넷 데이터 수집
	센싱 =>선세로 데이터 수집 
	RSS 리더/OPEN API => RSS 리더(웹사이트에서 업데이트 정보를 실시간으로 가져옴)
				   => OPEN API(미세먼지 센서)
	ETL => 추출 변환 적재 의 약자 

빅데이터 저장 기술 
	분산 파일 시스템 
		-네트워크 이용가능 -> 파일을 분산 시켜놓음 
	NOSQL 
		-스키마가 유연해 비관계형 DB에서 많이 쓰임
		-고속으로 데이터 처리 가능 
	병렬DBMS 
		-여러개의 프로세서를 가지고 데이터를 처리함 
		-굉장히 빠름
	네트워크 구성 저장 시스템 
		-종류가 다른 저장 장치를 하나의 서버에 연결해 총괄적으로 저장 및 관리하는 시스템 
		-NAS 

빅데이터 처리 기술 
	
	전처리 
		-> 데이터를 수집해서 정재해서 서비스가 가능한 상태로 만드는 과정
		->위에서 봤던 처리과정에서 중간에 축소 방식이 추가됨 
	실시간 처리
		-> 말 그대로 실시간 처리 은행 같은 곳에서 하는 계좌 관리 같은 것

대표적인 빅데이터 처리 기술 
	맵리듀스 - 구글에서 만든 대용량 데이터 처리 S/W
	하둡 - 맵리듀스를 오픈소스로 구현함 
	빅데이터 처리 기술은 상용보다는 오픈 소스의 비중이 높음

빅데이터 분석 기술 
	통계분석 

	-기술통계 - 평균,합 ,표준편차 
	-다변량 분석 - 2개 이상의 변수를 동시에 분석
	-회귀분석 - 변수들이 결과에 영향을 미치는 정도를 추정
	-시계열 분석 - 시간 변수의 흐름에 따른 종속변수의 움직임을 이해라고 예측하는 기법
	- 가장 많이 사용되는 도구 R,SAS   

	-데이터 마이닝
		- 통계분석은 가설 설정 -> 분석********************************    
		  데이터 마이닝은 -> 비계획적 데이터를 대상으로 한다는 차이 
		
		데이터 집합에서 의미 있는 데이터 추출 
		-> 목표 = 패턴 발견, 미래 예측  

	목표지향 = 명확한 목표 , 예측변수
	목표불명 = 데이터에 숨겨진 패턴을 분석

	- 연관성 규칙 알고리즘 
		상품의 연관 관계와 같은 특정 용도를 도출하기 위해 고안된 알고리즘  
	- 협업 필터링 -> 고객을 배재하고 상품만을 보고 측정 
			  -> 기존 데이터가 많아야 함 /실시간 기호 변화 예측 힘듬
